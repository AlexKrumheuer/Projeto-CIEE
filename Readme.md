Requisitos

- Criar um formulário para cadastrar animais (nome, descrição, data de nascimento, espécie, habitat e país de origem)
- Criar um formulário para cadastro de cuidados do animal (alimentação, exame veterinário, vacinação, treinamento) descrição para cada um desses campos e frequência que esses campos são realizados
-  Listar, Cadastrar, Atualizar e Remover animais e cuidados.
